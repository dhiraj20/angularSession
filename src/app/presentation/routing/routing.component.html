Routing
Routing is used for implementing navigation in Angular.

1. Import RouterModule, Routes

2. Configure the router via RouterModule.forRoot()

3. The RouterOutlet is a directive from the router library which acts as a placeholder that marks the spot in the template where the router should display the components for that outlet.
