Directives
Structural
Modify the structure of DOM ex. ngIf, ngFor
Attributes
Modify the attributes of DOM elements ex. ngClass, ngStyle
*ngIf: You can add or remove an element from the DOM by applying an NgIf directive to that element.
*ngFor: NgForOf is a repeater directive: a way to present a list of items.
*ngClass: NgClass typically control how elements appear by adding and removing CSS classes dynamically.
*ngStyle: NgStyle can set inline styles dynamically, based on the state of the component.
